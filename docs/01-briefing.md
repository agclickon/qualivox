# Briefing do Projeto

## Tipo de Projeto
SaaS

## Objetivo de Negócio
Criar um CRM de gestão de leads com integração ao WhatsApp e qualificação automática por IA para automatizar e otimizar o processo de vendas

## Público-Alvo
Pequenas empresas, consultores, agências de marketing, vendedores autônomos

## Nicho de Mercado
Qualquer segmento (solução universal)

## Escopo Desejado
CRM com integração WhatsApp via Evolution API, kanban para gestão de leads, qualificação automática por IA, chat interno, automação via N8N, dashboard com relatórios e métricas, gestão de equipes/usuários, automação de follow-up, integração com outras ferramentas, sistema de notificações, sistema de templates de mensagens, módulo de importação/exportação de leads, sistema de tags e categorização, histórico completo de interações, sistema de alertas, relatório de performance individual, sistema de backup, logs de auditoria, webhooks, configuração de horários de funcionamento

## Stack Tecnológica
Frontend: React/Next.js, Backend: Node.js, Banco de dados: PostgreSQL, Deploy: Vercel
