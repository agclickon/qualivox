# LeadFlow CRM

> CRM de gestão de leads com integração nativa ao WhatsApp e qualificação automática por IA.

**Versão atual:** `v1.5.0`

## Visão Geral

LeadFlow é uma plataforma SaaS de CRM voltada para pequenas empresas, consultores e agências que precisam gerenciar leads capturados via WhatsApp. Toda a comunicação acontece dentro da plataforma — sem precisar abrir o celular.

## Stack Tecnológica

### Frontend
- **Next.js 14** — App Router, Server Components, API Routes
- **React 18** — com hooks e SSE para tempo real
- **TypeScript 5** — tipagem estrita
- **Tailwind CSS 3** — utility-first styling
- **shadcn/ui** — componentes de UI acessíveis (Radix UI)
- **Recharts** — gráficos e visualizações
- **React Hook Form + Zod** — formulários com validação

### Backend
- **Next.js API Routes** — endpoints REST integrados
- **Prisma 5** — ORM com migrations
- **SQLite** — banco de dados local (via `prisma` com `better-sqlite3`)
- **JWT + bcrypt** — autenticação segura com refresh tokens

### WhatsApp
- **@whiskeysockets/baileys v7** — conexão direta ao WhatsApp Web (sem Evolution API)
- **Sessões persistidas** em `globalThis` (Map de sockets ativos)
- **QR Code** para autenticação de novas conexões
- **Multi-conexão** — suporte a múltiplos números simultâneos

### IA
- Módulo unificado `src/lib/ai-provider.ts` com suporte a:
  - **OpenAI** (GPT-4o, GPT-4o-mini, Whisper, text-embedding-3-small)
  - **Anthropic Claude** (claude-sonnet-4-6, claude-opus-4-6, etc.)
  - **Google Gemini** (gemini-2.0-flash, gemini-1.5-pro, text-embedding-004)
  - **xAI Grok** (grok-3, grok-3-mini, etc.)
  - **DeepSeek** (deepseek-chat, deepseek-reasoner, etc.)
- Provedor ativo configurável em **Configurações → Integrações** sem reiniciar o servidor
- **ElevenLabs** — síntese de voz por agente (velocidade, estabilidade, seleção de voz)
- Funções: qualificação de leads, geração de respostas, sumarização, transcrição de áudio, visão computacional, embeddings RAG
- **Zustand** — gerenciamento de estado global no cliente

### Automação
- **N8N** — workflows externos via webhook (URL configurável)

## Instalação e Configuração

### Pré-requisitos
- Node.js 18+
- npm ou yarn

### 1. Instalar dependências

```bash
npm install
```

### 2. Configurar variáveis de ambiente

Crie um arquivo `.env` na raiz:

```env
# Banco de dados
DATABASE_URL="file:./dev.db"

# Autenticação
JWT_SECRET="seu-secret-jwt-aqui"
JWT_REFRESH_SECRET="seu-refresh-secret-aqui"

# IA (opcional — pode configurar pela interface)
OPENAI_API_KEY=""
ANTHROPIC_API_KEY=""
GEMINI_API_KEY=""
GROK_API_KEY=""
DEEPSEEK_API_KEY=""

# N8N (opcional)
N8N_URL=""
N8N_API_KEY=""

# App
NEXT_PUBLIC_APP_URL="http://localhost:3000"
```

### 3. Inicializar banco de dados

```bash
npm run db:push       # Cria as tabelas
npm run db:seed       # Insere dados iniciais (usuário super_admin)
```

### 4. Rodar em desenvolvimento

```bash
npm run dev
```

Acesse: `http://localhost:3000`

**Login padrão após seed:**
- Email: `admin@leadflow.com`
- Senha: `admin123`

## Funcionalidades em Destaque

### WhatsApp Multi-conexão
- Múltiplos números conectados simultaneamente
- Autenticação via QR Code na interface
- Reconexão automática em caso de queda
- Suporte a mensagens de texto, imagem, áudio, vídeo, documentos, stickers
- Atendente padrão configurável por conexão

### Sistema de Atendentes
- Atribuição manual de conversas a atendentes
- Transferência com registro de auditoria (`ConversationTransfer`)
- Badge "Em espera" para conversas sem atendente atribuído (`status = "waiting"`)
- Histórico de atribuições na timeline do lead

### Timeline de Lead
- Registro automático de eventos: movimentações de pipeline, atribuições, transferências, follow-ups enviados
- Anotações manuais com data, hora e autor
- Edição/exclusão permitida apenas ao criador ou `super_admin`
- Filtros por tipo de evento

### IA Multi-provedor
- Troca de provedor em tempo real via configurações (sem reiniciar servidor)
- Qualificação automática: score 0-100, classificação quente/morno/frio
- Análise de sentimento, extração de orçamento/prazo/necessidades
- Geração de sugestão de resposta contextualizada
- Sumarização de conversas longas

### Sistema de Agentes IA
- Criação de agentes com personalidade, tom e prompt de sistema configurável
- Três modos de atuação: **autônomo**, **assistido** (humano aprova sugestão) e **híbrido**
- **RAG** — base de conhecimento por agente (PDF, TXT, DOCX, MD) com busca semântica por embeddings
- **Orquestrador** com sub-agentes especializados: qualificação, suporte e follow-up inteligente
- **Transcrição de áudio** via Whisper API — contexto completo para o agente
- **Visão computacional** — leitura de imagens enviadas pelo lead (GPT-4o / Gemini / Claude / Grok)
- **Síntese de voz** com ElevenLabs — agente pode responder com mensagem de áudio
- **RAG dinâmico** — indexação automática de conversas concluídas com revisão humana
- **Escalada inteligente** — regras configuráveis, alerta visual e notificação ao atendente
- **Dashboard de agentes** — métricas, taxa de escalada, satisfação estimada e tendências

### Google Calendar Integrado
- Integração nativa OAuth2 com Google Calendar por empresa (SaaS-ready)
- Agendamento automático por IA — agente detecta intenção e cria evento
- Views mês, semana e lista com navegação intuitiva
- Criação manual de eventos com Google Meet e envio de convites
- Card de eventos no painel da conversa
- Auto-renovação de tokens e sincronização bidirecional

### Conversão Lead → Cliente
- Conversão automática ao mover lead para `fechado_ganho` ou manual via botão
- Atualização automática do `lifecycleStage` para `cliente`
- Webhook enriquecido para CRMs externos (HubSpot, Pipedrive, etc.)
- Payload completo: lead + anotações + análise IA (sentimento, score, tópicos)
- Log de entregas com retry automático e backoff
- Assinatura HMAC-SHA256 para segurança
